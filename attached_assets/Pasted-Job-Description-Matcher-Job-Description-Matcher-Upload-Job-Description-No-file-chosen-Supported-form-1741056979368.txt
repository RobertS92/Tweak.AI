Job Title : AIML ( Python) Engineer
Job Location : Sunnyvale, CA or Austin, TX (Onsite)
Job Duration: Fulltime
 
Role :

• Developing production-ready web applications and machine learning pipelines
• Deploying, administering, and maintaining ML applications and supporting infrastructure in AWS or other cloud/on-prem infrastructure
• Setting up scalable ML infrastructure for training and monitoring predictive models
• Implementing self-hosted LLM inferencing services on AWS
• Supporting Kubernetes-based workloads on AWS EKS Roles.
• Implementing best practices for CI/CD, monitoring, and telemetry
• Ensuring production security and compliance by working within predefined access controls

MLOps:
